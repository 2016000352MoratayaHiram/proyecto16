Conceptos Básicos

  • Dominio: Nombre único que identifica un sitio en Internet.
  • Subdominio: División del dominio principal, como "blog.ejemplo.com".
  • URL: Dirección completa de una página web.
  • DNS: Sistema que traduce nombres de dominio a direcciones IP.
  • Hosting: Servicio que almacena sitios web en servidores.
  • FTP: Protocolo para transferir archivos a través de Internet.
  • HTTP: Protocolo de transferencia de hipertexto.
  • HTTPS: Versión segura de HTTP.
  • W3C: Consorcio que desarrolla estándares web.
  • UTF-8: Formato de codificación de caracteres.
  • HTML5: Última versión del lenguaje HTML.
